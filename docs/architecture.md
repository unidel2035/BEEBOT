## Слайд 1. Что такое BEEBOT?

**BEEBOT** — цифровой помощник пчеловода Александра Дмитрова.

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   Подписчик в Telegram                          │
│       ↓                                         │
│   "Как принимать пергу?"                        │
│       ↓                                         │
│   BEEBOT отвечает голосом Александра,           │
│   прикладывает PDF-инструкцию,                  │
│   предлагает купить продукт                     │
│                                                 │
└─────────────────────────────────────────────────┘
```

**Три роли бота:**
- Консультант — отвечает на вопросы о продуктах пчеловодства
- Логист — принимает заказы, собирает адрес, считает доставку
- Аналитик — показывает статистику продаж пчеловоду

```
│       ▼                                                │

│                                                        │

```

**Зачем SSH-туннель?**
Groq API блокирует IP дешёвых VPS. Решение: запросы идут через домашний сервер (hive), у которого IP не заблокирован.

**systemd-сервисы на hive:**
- `groq-proxy.service` — reverse proxy для Groq API
- `groq-tunnel.service` — SSH-туннель VPS ↔ hive (авто-рестарт)
