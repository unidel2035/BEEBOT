### ✅ 3.4 Долгосрочная память — выполнено (25.03.2026)

**Реализовано:**
- SQLite (`data/memory.db`) — хранение фактов о пользователях между сессиями
- `src/memory.py` — `UserMemory`: `add_fact`, `get_facts`, TTL 30 дней
- Автоизвлечение фактов из диалога: `extract_fact(query, response)` через LLM
- Внедрение в оркестратор: `memory_facts` добавляются в контекст перед ответом
- `/teach <факт>` в `src/admin.py` — добавление знания в KB напрямую

**Онтологический слой Integram (выполнено 25.03.2026):**
- Таблица **Симптомы** (6137): 12 симптомов с ключевыми словами
- Таблица **Показания к применению** (6143): 17 связок «Симптом → Товар + дозировка»
- Таблица **Источники знаний** (6150): 14 записей «Товар → KB-ключ»
- `src/ontology.py` — `OntologyCache`: загружает из Integram при старте
- `match(text)` — находит симптомы в запросе → вставляет рекомендацию в `memory_facts`

### 3.7 Профиль здоровья — мост SQLite → Integram (P2)

**Идея:** Когда `extract_fact` сохраняет факт о здоровье → дублировать в Integram «Профиль здоровья». Пчеловод видит CRM-карточку клиента с историей здоровья.

**Задачи:**
- [ ] Определить health-категорию факта (болезнь, симптом, аллергия)
- [ ] При `add_fact`: если health-факт + telegram_id есть в CRM → создать/обновить запись в «Профиль здоровья»
- [ ] Поиск клиента по telegram_id через `integram_client.py`

## Фаза 5: Экосистема — Подключение к AgentBus

### 5.1 Регистрация в AgentBus (P3 ↓)

Система памяти теперь самодостаточна (SQLite + Integram). Приоритет снижен.

**Задачи:**
- [ ] При старте бота: POST `/api/agent-bus/register` → `{agentId: "beebot", capabilities: ["crm", "kb-search", "orders"]}`
- [ ] Обработчик входящих сообщений: `/api/agent-bus/inbox/beebot`
- [ ] Фоновая задача: heartbeat каждые 30 сек
- [ ] API-инструменты для других агентов: поиск по KB, получение статуса заказа

### 5.2 Общий профиль пользователя (P3 ↓, переформулировать)

Частично закрыт через SQLite (факты) + Integram «Профиль здоровья» (задача 3.7). Подключение к Buddy создаёт третье хранилище без явной пользы для пчеловода.

- [ ] Оценить: достаточно ли SQLite + Integram для персонализации?
- [ ] Если нет — рассмотреть синхронизацию с `fst` только для межагентного обмена
